# Data Sources

This document defines preferred primary data sources.

## Market Data
- Equity prices: Exchange feeds, issuer IR
- Index data: S&P Dow Jones Indices
- Commodities: CME, LME

## Macroeconomic Data
- Employment: U.S. Bureau of Labor Statistics (BLS)
- Inflation / PPI: BLS
- GDP: U.S. Bureau of Economic Analysis (BEA)
- Housing: U.S. Census Bureau

## Credit & Rates
- Yield curves: U.S. Treasury
- Credit spreads: ICE/BofA indices (HYG/LQD proxies)

## Validation Rule
Secondary sources (media, commentary) are never authoritative.
